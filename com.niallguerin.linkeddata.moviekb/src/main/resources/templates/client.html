Welcome to the Movie Actor SPARQL Client

This is a simple SPARQL web application that lets you search for actors, find their film lists, and view their profile information, images, and awards (if any). It is powered by Linked Data Services using SPARQL, DBPEDIA, Wikidata, and RDF4J framework.

To search, enter first and last name of an actor e.g. Daniel Craig, Jodie Foster.

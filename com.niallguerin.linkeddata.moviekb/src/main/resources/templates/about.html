This web application uses single and federated SPARQL queries to obtain information about actors from favorite movies.

It allows search by actor name, expecting first and last name as a search string. It then provides profile biography data, photos (if available), and awards won by the actor.

The objective is to apply the principles from class lectures with a focus on SPARQL endpoints embedded within a Spring Boot web application.

The application does not store data locally, but the RDF4J framework was used to build the application, and an in-memory graph database can be enabled.
